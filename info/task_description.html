The child of your nearby friends has started to learn math not long ago. Now he is learning a simple math operations as addition and subtraction. But soon he will start to learn more complex themes: multiplication, division, powers and roots.
You want to help him so your task is to create a two calculators - SimpleCalc() Ð¸ ComplexCalc(). The simple one will use only 2 operations - addition and subtraction with methods add(*args) and sub(x, y).
add(*args) - count the sum of the all arguments.
sub(x, y) - count (x - y).
The complex one will be the decorator of the simple and will add additional methods/operations:

multi(*args) - count the product of the all arguments.
div(x, y) - count x/y.
power(x, y) - count x in the power of y.
root(x, y) - count root of degree y from x.

If the result is an integer number - return it as int() type.
Round all float results to the 5 decimal digits.
In this mission you should use the Decorator design pattern.

Examples: 

calculator = SimpleCalculator()

calculator.add(9, 1, 5, 5, 0.2) == 20.2
calculator.sub(54, -16) == 70
    
calculator = ComplexCalculator(calculator)
    
calculator.multi(1, 3, 3, 9) == 81
calculator.div(3, 9) == 0.33333
calculator.power(2, 10) == 1024
calculator.root(81, 4) == 3


Input: numbers and name of the math operation as the class method.

Output: the result of the operations.

How it is used: For work with math fuctions and objects.

Precondition: Real numbers.
